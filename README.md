# WATF-for-few-shot-learning
Here is the code and datasets of WATF for few shot image classification.I provide a PyTorch implementation of WATF. 
Prerequisites:
        Linux
        Python 3.8
        Pytorch 1.7.0
        GPU + CUDA CuDNN
        pillow, torchvision, scipy, numpy
